# Screenshots

This folder contains screenshots demonstrating the StayEasy RAG system.

## Screenshots Included:

### 1. Web Interface (Gradio UI)
**File:** `web-interface.png` or `gradio-ui.png`
- Shows the StayEasy RAG Customer Support Assistant web interface
- Displays chat interface with Q&A examples
- Shows retrieved sources panel with document references
- Demonstrates the RAG system in action with real queries

### 2. Evaluation Metrics Table
**File:** `evaluation-metrics.png` or `metrics-table.png`
- Shows detailed evaluation metrics with scores and contributions
- Displays: Retrieval Accuracy (100%), MRR (0.95), Avg Relevance (5.00/5), Avg Correctness (4.90/5), Avg Faithfulness (5.00/5)
- Overall score: 98.6/100

### 3. Performance Improvement Report
**File:** `performance-improvement.png` or `optimization-results.png`
- Shows before/after comparison of system improvements
- Documents changes made to ingest.py, answer.py, and evaluate.py
- Displays performance improvement from 96.5/100 to 99.5/100
- Highlights markdown-aware chunking and prompt improvements

### 4. Evaluation Results Summary
**File:** `evaluation-results.png` or `rag-score.png`
- Shows overall RAG system score: 96.5/100
- Breakdown of metrics: Retrieval Accuracy (100%), Answer Relevance (4.80/5), Answer Correctness (4.60/5), Faithfulness (4.90/5)

## How to Add Screenshots:

1. Save your screenshot images in this folder
2. Name them according to the suggestions above
3. Supported formats: PNG, JPG, JPEG
4. Commit and push to GitHub:
   ```bash
   git add screenshots/*.png screenshots/*.jpg
   git commit -m "Add project screenshots"
   git push
   ```
