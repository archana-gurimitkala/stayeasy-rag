# StayEasy Pricing and Fees

## Guest Fees

### Service Fee
- 10-14% of booking subtotal
- Covers 24/7 support, secure payments, platform costs
- Shown before you book

### Cleaning Fee
- Set by host
- One-time fee per stay
- Shown in price breakdown

### Security Deposit
- Some hosts require this
- Typically $100-$500
- Fully refundable if no damage

### Extra Guest Fee
- Some hosts charge for guests beyond a certain number
- Example: $20/night for each guest after 2

## Host Fees

### Service Fee
- 3% of booking subtotal
- Deducted from payout automatically

### No listing fee
- Free to create and publish listings
- You only pay when you get bookings

## Price Breakdown Example

**Guest View:**
```
Nightly rate: $150 x 3 nights = $450
Cleaning fee: $75
Service fee (12%): $63
Total: $588
```

**Host Payout:**
```
Nightly rate: $150 x 3 nights = $450
Cleaning fee: $75
Subtotal: $525
Host fee (3%): -$15.75
You receive: $509.25
```

## Currency
- Prices shown in your local currency
- Conversion rates updated daily
- No currency conversion fee for guests

## Taxes
- Local taxes may apply depending on location
- Some regions require hosts to collect occupancy tax
- Tax amounts shown separately in booking
